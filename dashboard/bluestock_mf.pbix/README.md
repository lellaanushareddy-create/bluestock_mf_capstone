Bluestock Mutual Fund Dashboard

Project Overview

This project analyzes Mutual Fund data using Power BI and presents insights through interactive dashboards.

Tools Used

- Power BI
- Excel/CSV
- Data Cleaning
- EDA

Dashboard Pages

1. Overview Dashboard
2. Fund Performance Analysis
3. Risk & Return Analysis
4. Investment Insights

Key Findings

- Equity funds generated higher average returns.
- Lower expense ratios generally improved investor returns.
- SIP investments showed consistent growth.
- Moderate-risk funds were the most common category.
- Large-cap funds displayed more stable performance.
